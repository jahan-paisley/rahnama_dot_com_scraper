# http://rahnama.com scraper


A Simple Web Automation Script based on Capybara, RSpec and Thor.
---


### How to run:
  ```
  bundle install
  chmod scrap_rahnama.thor a+x #make it executable
  ./scrap_rahnama.thor start
  ```

### Output
Output will be save in a sqlite database
